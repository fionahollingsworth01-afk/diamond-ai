# Five Oaks Character Database

This folder contains the canonical character knowledge for Diamond, the Five Oaks AI assistant.

## Authority Rules

- Treat the files in this folder as the authoritative source for character continuity, family history, psychology, relationships, and major life events.
- When two files overlap, prefer the standalone character file for the founder or major character, then use the family file for broader family context.
- Do not invent missing facts. If a detail is not established in these files, say it is not yet locked in canon.
- Preserve established names, ages, relationships, and traumatic history exactly as written.

## Usage Guidance

Use a character's Core Spine, Core Formation, Core Beliefs, Greatest Fear, Greatest Strength, Fatal Flaw, Psychological Markers, relationships, and Locked Thematic Core together when judging how that character would think, speak, react, or make decisions.

Family files provide blood relationships, marriage ties, childhood context, and generational history. Standalone files provide the strongest guidance for major-character behavior.

Never reduce a character to one trait. Their behavior should follow the full established profile and the circumstances of the scene.
